Jugando con tambores

Este es un juego interactivo para reproducir ritmos del Piano. Para interactuar presiona las teclas : Flecha izquierda, derecha, escape y enter! Presiona enter para comenzar!

¿Querés jugar en el modo audio asistido?

Este modo esta pensado para aquellas personas que tengan dificultad para leer, baja o nula visibilidad o bien quien quiera una dificultad menor!
